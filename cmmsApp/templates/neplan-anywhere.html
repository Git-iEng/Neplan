Overview

NEPLAN ANYWHERE - browser-based tool for operations, monitoring, connection management & request assessments

NEPLAN Anywhere is a browser-based add-on tool with a user-friendly, simplified graphical interface, designed to support the day-to-day tasks of utilities. It complements the full NEPLAN 10 / 360 planning tools and serves as a lighter, always-accessible interface for operations, monitoring, connection-management, and request assessments.

  • Geodata & Smart-Meter Integration

    Integration of GIS data and smart-meter data via MDM systems for improved accuracy.

  • Online MV/LV Monitoring

    Real-time tracking of voltages, loads, and other performance metrics for medium- and low-voltage networks.

  • Connection Request Management

    Assessment & management of solar PV or EV-charging connection requests with ERP integration.

  • Mobile-Friendly Interface

    Supports inspections, approvals, and assessments on-site via tablets or smartphones.

  • Data Hub & Integration

    Acts as a data hub via web services, enabling data sharing with external systems (GIS, MDM, ERP).

Key Use Cases

Where NEPLAN Anywhere Delivers Value

Browser-based tools for day-to-day utility work: monitor, assess connections, make field decisions, and integrate data.

  • Online Monitoring

    Retrieve smart-meter / measurement data and visualize live or near-real-time metrics (voltages, loadings, etc.); detect limit violations in LV/MV networks and generate alerts.

  • Connection Request Assessment

    Evaluate feasibility of new connections (e.g., solar PV, EV chargers) under current network conditions; provide quick feedback (green/yellow/red) based on voltage and thermal loading criteria-saving time for utilities and applicants.

  • Connection Request Management

    Track requests end-to-end (submitted, pending, approved, rejected). Import or update from ERP systems; maintain time-stamped records and archive assessments for future reference.

  • Mobile / On-site Decision Making

    Use tablets or smartphones to view network topology, run basic analyses, and complete assessments on the spot-ideal when permissions or decisions must be made outside the office.

  • Integration with Other Systems

    Act as a data interface (web services) with GIS, MDM, ERP, SCADA, etc., to share and update data across systems-reducing duplication and improving consistency.

Why Use NEPLAN Anywhere

  • Speeds up routine operations (approving connections, monitoring networks) without needing full-feature software client.
  • Improves responsiveness: real-time visibility and decision making, fewer delays.
  • Empowers field teams and non-specialists to make informed decisions with minimal training.
  • Increases transparency and customer satisfaction (for example via customer portals for connection assessment).
  • Helps avoid network overloads or violations by catching issues early through monitoring.
