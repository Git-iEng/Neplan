Overview

NEPLAN V10 | NEPLAN 360 - On-Premises& Cloud Solutions

NEPLAN V10 | NEPLAN 360 offers flexible deployment options for modern energy networks, combining the reliability of on-premise solutions with the scalability and accessibility of the cloud. NEPLAN 360 runs entirely in the cloud, providing instant access anywhere via the NEPLAN app, centralized maintenance, and simplified IT workflows-ideal for multi-user, multi-site, or multinational companies. For organizations requiring full control over data, NEPLAN V10 delivers the same powerful features through on-premise or intranet installations with full MS-SQL database support, GIS map server integration, and user-specific study case management.

Both solutions share identical capabilities for network analysis, planning, optimization, and simulation, ensuring a consistent experience across cloud and on-premisesenvironments. With Web Service integration, NEPLAN V10/360 connects seamlessly to GIS, SCADA, and Smart Grid applications, enabling automation, real-time data exchange, and SaaS-style deployment for future-ready energy enterprises.

  • Cloud-First NEPLAN 360

    NEPLAN 360 runs in the cloud with no desktop installation needed. Accessible from anywhere with the same version for all users, it simplifies maintenance for enterprises and supports modern licensing models like time-based usage.

  • On-Premise NEPLAN V10

    Installed on your servers with MS-SQL multi-user database support, user-specific study cases, GIS map integration, and local file storage options-designed for large networks and enterprises with strict data control needs.

  • Unified Capabilities

    Both cloud and on-premises versions deliver identical functionality-load flow, planning, optimization, and simulations-ensuring consistency whether hosted on a private intranet or in the cloud.

  • Web Service Integration

    NEPLAN V10/360 provides Web Services for integration with GIS, SCADA, and Smart Grid applications, enabling automation, real-time data exchange, and SaaS-style connectivity.

  • Enterprise Scalability

    Ideal for multi-user, multi-site, or multinational environments-no version conflicts, centralized maintenance, and simplified IT workflows for energy enterprises.
