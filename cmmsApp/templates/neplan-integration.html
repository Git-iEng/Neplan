Overview

NEPLAN Add-ons - modular, flexible & project-specific solutions

The NEPLAN OEM / SaaS WebService offers direct access to NEPLAN V10/360’s powerful calculation modules through secure web service APIs, eliminating the need for a traditional graphical interface. GIS, SCADA, and Smart Grid application providers can seamlessly integrate NEPLAN’s analysis, planning, and simulation capabilities into their platforms, enabling network creation, real-time control, automated calculations, and result retrieval directly within their existing workflows.

Designed for flexibility and scalability, the WebService supports both cloud and intranet deployments, ensuring compatibility with various IT infrastructures. It also integrates with advanced NEPLAN add-ons such as CGMES, Real-Time Data, Load Modelling, and Data Management, empowering utilities and grid operators to automate processes, perform predictive simulations, and maintain secure, reliable, and future-ready energy networks.

  • CGMES 2.4.15 Add-on

    Enables automatic load flow time simulation, cyclic data import/export, and CIM-based compliance for DSO-TSO data exchange.

  • Real-Time Data Add-on

    Integrates with control systems, imports live data and switching states, and supports predictive load flow calculations.

  • Load Modelling Add-on

    Generates realistic load scenarios, integrates GIS/SAP data, and supports flexible parameter tuning for simulations.

  • Data Management Add-on

    Supports incremental data imports, network optimization, and automated data model preparation for large systems.

  • GIS Integration

    Run NEPLAN network analysis directly within GIS platforms for powerful visualization and optimization workflows.
